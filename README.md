# Ad Filters

A collection of filter lists for ad blockers

## How to subscribe

To subscribe to a filter, two solutions

1. (recommended) Do right-click on a link on this list, then "Subscribe to filter list..."
2. Open the filter file in GitHub, get it's raw link and add it to your ad blocker
